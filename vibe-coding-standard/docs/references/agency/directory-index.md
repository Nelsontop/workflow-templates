# Agency Directory Index

本文件是 `docs/references/agency` 的手写目录索引，按分类列出可读取的 Markdown 参考文档。

## 使用建议

- 先看分类，再进入具体文档，不要一次性读取整类内容。
- 优先选择与当前任务最接近的 1 到 3 个文档。
- 如果本地技能已明确命中，优先使用本地技能，不额外加载这里的参考。
